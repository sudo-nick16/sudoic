Commands Description
?p / ?play <song_name / <youtube_url> To play a song. (Note : if a song is already playing, then requested song will be added to the queue.)
Ex: ?p Shadow of the sun or ?p https://www.youtube.com/watch?v=pBru4MPtEik
?p / ?pl <yt_playlist_url> To play songs from a playlist.
Ex: ?p https://www.youtube.com/playlist?list=PL4PY_vHKZgwjrdBtYVL6tjfgUAiNR0xk8
?s / ?fs To skip the current song. (Note : use these commands alternately to skip, entering same command will only work once.)  Ex: ?s
?v To pause the song.
Ex: ?v
?r To resume the song.
Ex: ?r
?stop To clear queue and stop playing songs.
Ex: ?stop
